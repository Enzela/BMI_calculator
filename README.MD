# BMI Calculator

A simple and intuitive Body Mass Index (BMI) calculator that helps users assess their body weight category based on height and weight measurements.


## 📋 Features

- **Calculate BMI instantly** - Get your BMI number with just height and weight inputs
- **Dual unit support** - Works with both Metric (kg/cm) and Imperial (lbs/in) units
- **Instant categorization** - Automatically shows which weight category you fall into
- **Health recommendations** - Provides basic guidance based on your results
- **User-friendly interface** - Clean, simple design for easy use
- **Mobile responsive** - Works seamlessly on desktop, tablet, and mobile devices
